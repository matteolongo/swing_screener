# Degiro Order Setup: Breakout vs Pullback

This guide explains how to execute setup types manually in Degiro while keeping the same risk plan generated by Swing Screener.

## Breakout Setup

Use when price still needs to break above a trigger level.

1. Entry intent:
- Use `BUY STOP` in Degiro slightly above the breakout trigger.

2. Stop placement:
- Keep stop below the invalidation level from Swing Screener.

3. Position sizing:
- Keep shares consistent with the risk plan shown in the app.

4. Invalidation:
- If breakout never triggers and price weakens, cancel/skip instead of chasing.

### Breakout Checklist

- Trigger level identified
- BUY STOP set above trigger
- Stop below invalidation
- Risk amount still within plan

## Pullback Setup

Use when trend is intact and entry is planned on a dip/reclaim area.

1. Entry intent:
- Use `BUY LIMIT` near pullback/reclaim zone.

2. Stop placement:
- Keep stop below pullback invalidation (swing/support level).

3. Position sizing:
- Keep shares aligned to your planned risk.

4. Invalidation:
- If price runs without filling, do not move the limit up to force entry.

### Pullback Checklist

- Pullback/reclaim level identified
- BUY LIMIT placed at planned zone
- Stop below invalidation
- Risk amount still within plan

## Common Mistakes

- Chasing breakout candles with market orders after large spikes.
- Raising pullback limits repeatedly just to get filled.
- Changing stop placement in Degiro without recalculating risk.
- Sending multiple entry orders for the same ticker and over-sizing the trade.

## Practical Rule

- Breakout logic: condition-first entry (`BUY STOP`).
- Pullback logic: price-improvement entry (`BUY LIMIT`).

If setup context is unclear, skip the trade and wait for a clean signal.
